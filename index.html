Hello world

Gaudiamus base setup

In the scss folder, you will find 3 files:

  • index.scss
  • my-vars.scss
  • compositions.scss

my-vars.scss contains all variables you can overwrite (commented out)
compositions.scss is basically "your-style". It is an example of how to build your own style guide.
index.scss takes care of loading files in the correct order and the only file that needs to be compiled.

That's it. It's that simple!

run "npm run dev"
